# CORTEX LM

CORTEX is a local AI configuration built on Qwen2.5 7B using Ollama.

It focuses on:
- Blunt, honest responses
- No emotional padding
- General-purpose use (coding, logic, chat)
- Clear acceptance of shutdown and user control

This repository contains:
- A Modelfile
- Developer notes (licensing resides in these)

Model weights are not included.
